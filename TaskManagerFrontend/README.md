# Employee Task Frontend

React + Vite + Tailwind frontend for your Employee/Task API.

## Setup

```bash
npm install
npm run dev
```

Backend is expected at `http://localhost:8080` with endpoints:

- POST `/employees/login`
- POST `/employees/signup`
- GET `/employees`
- GET `/employees/{id}`
- GET `/employees/{id}/tasks`
- CRUD `/tasks`
- GET `/tasks/{id}/workflows`
- PUT `/tasks/status`
```
